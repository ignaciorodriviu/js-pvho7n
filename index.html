¿Qué gráfico es el más adecuado?

Para este tipo de casos donde tenemos objetos con valores parecidos, descartaríamos el gráfico de Sectores respecto a los de barras y columnas, dado que es difícil diferenciarlos. Como se desea representar la comparación entre objetos, con una variable por objeto, con pocas categorías y muchos datos, el gráifico más adecuado es el 4.
